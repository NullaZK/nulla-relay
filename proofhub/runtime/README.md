# ProofHub Runtime

ℹ️ The ProofHub runtime implements privacy-preserving proof verification using Pedersen commitments,
Merkle trees, and range proofs. It is built as a Substrate parachain runtime.

💁 The runtime is constructed using ready-made FRAME pallets from
[Polkadot SDK](https://github.com/paritytech/polkadot-sdk), plus custom pallets for proof verification
([pallet-proofs](../pallets/proofs/README.md)).

👉 Learn more about FRAME
[here](https://paritytech.github.io/polkadot-sdk/master/polkadot_sdk_docs/polkadot_sdk/frame_runtime/index.html).
